# Agentic Design Engineering

A Claude Code plugin that transforms generic UIs into intentional, inhabited digital places — using four interconnected design frameworks: CLEAR, PLACE, ALIVE, and VOICE.

## Repository Structure

```
agentic-design-engineering/
└── plugins/
    └── agentic-design-engineering/
        ├── README.md              ← Full documentation
        ├── CLAUDE.md              ← Plugin configuration
        ├── AGENTS.md              ← Agent coordination
        ├── skills/
        │   ├── ade-clear/         ← /ade:clear — structural foundations
        │   ├── ade-place/         ← /ade:place — atmosphere & metaphor
        │   ├── ade-alive/         ← /ade:alive — vitality & easter eggs
        │   ├── ade-voice/         ← /ade:voice — copy & tone
        │   ├── ade-audit/         ← /ade:audit — score all frameworks
        │   └── ade-transform/     ← /ade:transform — full pipeline
        ├── agents/
        │   ├── review/            ← clear-auditor, place-auditor, alive-auditor
        │   ├── design/            ← atmosphere-builder, metaphor-discoverer, vitality-injector
        │   └── voice/             ← akshansh-voice
        └── docs/
            ├── README.md
            └── decision-log-template.md
```

## Installation

```
claude plugin add akshansh/agentic-design-engineering
```

For full documentation — frameworks, slash commands, philosophy, and usage — see the [plugin README](plugins/agentic-design-engineering/README.md).
